# Kruskal's Algorithm (Minimum Spanning Tree) in Python
Kruskal's algorithm is a greedy algorithm used to find the Minimum Spanning Tree (MST) of a connected, weighted undirected graph. It sorts all edges by weight and adds the smallest edge (no cycles created). The time complexity for this algorithm is O(E log E) - E is the number of edges in the graph.
## Run
```
python main.py
```
